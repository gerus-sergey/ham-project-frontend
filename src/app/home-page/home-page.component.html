Главная страница

Данный сайт позволяет пользователю оценивать альтернативы относительно набора критериев с помощью метода гибридных оценок. Нахождение предпочтительной альтернативы производится посредством оценки критериев относительно друг друга и дальнейшей оценки набора альтернтив по критериям.

Для нахождения предпочтительной альтернативы пользователю необходимо создать наборы критериев и альтернатив и приступить к оценке. Также данный сайт поддерживает возможность оценивать альтернативы группой экспертов. Для этого необходимо создать предопределённые наборы критериев и альтернатив и назначить их пользователям. Для дальнейшей работы с сайтом пройдите регистрацию либо авторизуйтесь.
